OSU Men's Basketball Stats Dashboard

By Team Seven: Kamren Z. Glen Q. Mohamed M. Rith R.

About Project: The goal of this project was to scrape the OSU MBB Roster page found here. Some players' personal pages contain a "STATS" tab with their personal statistics. This project scrapes the main roster page, checks which players have a "STATS" tab on their personal page, and scrapes the statistics for each game played.

Data is then compiled into various graphs showing different aggregated statistics. For details on some of the calculations used, please see the About Game Scores section.

About Game Scores: When devising the concept for this project, our group wished for a method to use player statistics to compare performance; we envisioned a ranking system similar to OVR ranking in NBA 2K games. After some searching, we discovered a calculated statistic called "Game Score" conceived by basketball writer/analyst John Hollinger. The stat is meant to generalize a player's performance in a single game, and was created to be compared on a scale similar to points: 20 is a good score whereas a single digit score represents a less productive game. Details about the calculation of Game Score can be found here. Our project automatically calculates this statistic after reading the data for each game.
